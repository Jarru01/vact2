# Repository containing files (mainly cloudformation/heat scripts) from virtualization and cloud technologies 2 lectures at the university.
* aws1 - AWS cloud basics + automation tools + versioning
* aws2 - VPC, securing network (SG, NACL), VPC peering, S2S VPN, Direct Connect, PrivateLink, Transit Gateway
* aws3 - EC2 instances & EC2 autoscaling & load balancing
* aws4 - AWS Lambda, AWS Step Functions, AWS API GW, Containers (aws activity)
* aws5 - AWS CloudFormation – S3, CloudWatch
* openstack1 - Basic HEAT scripts (CloudFormation for openstack) 
* openstack2 - Advanced HEAT scripts
* terraform - basic terraform/**tofu** script
